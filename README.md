pjson
=====

Like `python -mjson.tool` but with moar colors (and less conf)

### Usage

    ⚡ echo '{"json":"obj"}' | pjson
    {
      "json": "obj"
    }

### Example With Curl

    ⚡ curl https://api.github.com/users/igorgue | pjson

### Install

Install `pygments` and `xmlformatter`:

    ⚡ pip install pjson xmlformatter
